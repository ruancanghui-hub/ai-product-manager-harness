# AI-PM H5 抽卡游戏设计规格

## 目标

将 `output/dragon-ball-role-art` 中 43 张角色图制作成可在手机浏览器使用的 H5 抽卡游戏。产品只做抽卡、角色技能揭示、收藏与技能包图鉴，不做卡牌对战。

## 已确认体验

- 抽卡：首页为天空岛召唤祭坛，点击“抽取角色”后翻牌。
- 揭示：展示角色全图、角色名、岗位、所属团队与三项核心技能。
- 收藏：已抽取角色写入本地存储，可随时回看。
- 技能包：展示 7 个团队分布、43 张卡牌、关键词搜索与团队筛选。
- 导航：底部固定“抽卡 / 收藏 / 技能包”。

## 视觉系统

- 参考图：`.superpowers/brainstorm/86891-1782612965/content/combined-system-target.png`。
- 气质：原创深色奇幻卡牌 UI、金属金边、羊皮纸信息面板、紫色选中态、电影级雷暴天空岛背景。
- 图像：角色卡必须使用现有 43 张真实角色图，不使用占位图。
- 字体：中文使用系统黑体；标题使用宋体/衬线字体栈体现史诗感。
- 主色：深夜蓝 `#071326`、金色 `#d6a84a`、羊皮纸 `#ead8b0`、紫色 `#7650c7`。
- 动效：抽卡翻转、揭示光晕、按钮按压；遵循 `prefers-reduced-motion`。

## 数据与状态

- 每张卡包含：ID、角色名、岗位、团队、图片路径、三项技能及技能说明。
- 收藏数据使用带版本号的 `localStorage` 结构。
- 抽卡优先从未收藏角色中随机；集齐后从全卡池随机。
- 搜索支持角色名、岗位、团队名与技能名。

## 响应式

- 主要基线：390 × 844 手机竖屏。
- 桌面端：内容限制在 1180px 内，抽卡页使用居中舞台，图鉴使用自适应网格。
- 所有触控目标不小于 44px；底部导航避让安全区。

## 验收标准

1. 43 张图片全部能加载，不出现破图。
2. 可以抽取、翻牌、查看三项技能并收入收藏。
3. 刷新页面后收藏仍在。
4. 技能包可以按团队筛选和关键词搜索。
5. 手机与桌面尺寸均无横向溢出、主要文字不截断。
6. 构建、单元测试、核心浏览器流程全部通过。
