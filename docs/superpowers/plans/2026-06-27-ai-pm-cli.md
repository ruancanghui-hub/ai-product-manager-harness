# AI PM Cross-Agent CLI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a globally installable `ai-pm` npm CLI that initializes and diagnoses Claude Code, Codex, and Cursor projects from one canonical skill source.

**Architecture:** Use dependency-free Node.js ESM modules with `node:test`. Keep canonical skills and portable templates under `package/`; convert CLI input into a deterministic installation plan, then execute that plan with conflict checks and rollback. Codex and Cursor share `.agents/skills`, while Claude receives `.claude/skills` plus native hooks.

**Tech Stack:** Node.js 20+, ESM, built-in `node:fs`, `node:path`, `node:child_process`, `node:test`, npm package format.

---
