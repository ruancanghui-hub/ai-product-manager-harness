# Porter's Five Forces (波特五力分析)

## 目的

通过分析行业内的五种竞争力量，评估行业吸引力和盈利潜力，为竞争战略和进入/退出决策提供依据。

## 核心概念

```
                    现有竞争者竞争程度
                   (Rivalry Among Existing Competitors)
                     - 竞争者数量与规模
                     - 行业增长率
                     - 固定成本高低
                     - 产品差异化程度
                     - 退出壁垒
                              │
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
        │                     │                     │
供应商议价能力              行业内竞争            买方议价能力
(Supplier Power)           (Industry)            (Buyer Power)
        │                     │                     │
        │                     │                     │
        └─────────────────────┼─────────────────────┘
                              │
                              │
              ┌───────────────┴───────────────┐
              │                               │
        替代品威胁                       新进入者威胁
    (Threat of Substitutes)          (Threat of New Entrants)
        - 替代品价格                    - 进入壁垒高低
        - 转换成本                      - 规模经济
        - 购买者意愿                    - 品牌忠诚度
                                        - 资本要求
```

**五力强度**:
- **高强度**：行业盈利能力低，竞争激烈
- **低强度**：行业盈利能力高，竞争温和

## 何时使用

- ✅ 进入新行业前，评估行业吸引力
- ✅ 制定竞争战略，找到差异化方向
- ✅ 投资决策，评估行业盈利潜力
- ✅ 定期战略审视，监控竞争环境变化
- ✅ 行业分析报告，理解行业结构

## 使用步骤

### Step 1: 定义行业边界

**问题**: 我们要分析哪个行业？行业边界如何界定？

**格式**:
```
行业定义：
- 行业名称：[行业名称]
- 行业范围：[地理范围/产品范围/客户范围]
- 主要产品/服务：[产品/服务列表]
- 主要客户群：[客户群描述]

价值链定位：
- 上游供应商：[供应商类型]
- 中游：[我们所在的环节]
- 下游客户：[客户类型]
```

**检查点**:
- [ ] 行业定义是否清晰？
- [ ] 边界是否合理（不过宽、不过窄）？
- [ ] 是否明确了地理范围？

### Step 2: 分析现有竞争者竞争程度（Rivalry Among Existing Competitors）

**问题**: 行业内现有竞争者之间的竞争有多激烈？

**分析维度**:
- **竞争者数量与规模**：竞争者多寡、规模分布
- **行业增长率**：高增长vs低增长
- **固定成本高低**：高固定成本导致价格竞争
- **产品差异化程度**：差异化低导致价格竞争
- **退出壁垒**：高退出壁垒导致过度竞争
- **战略利益**：战略重要性高的市场竞争更激烈

**格式**:
```
现有竞争者竞争程度：

竞争强度：[高强度/中强度/低强度]

主要竞争者：
1. [竞争者1]
   - 市场份额：[XX%]
   - 竞争优势：[优势描述]
   - 竞争策略：[策略描述]

2. [竞争者2]
   - 市场份额：[XX%]
   - 竞争优势：[优势描述]
   - 竞争策略：[策略描述]

竞争特征：
- 竞争者数量：[多/中/少]
- 行业增长率：[高/中/低]
- 固定成本：[高/中/低]
- 产品差异化：[高/中/低]
- 退出壁垒：[高/中/低]

竞争态势：
- 价格竞争：[激烈/温和/无]
- 非价格竞争：[品牌/渠道/技术/服务...]
- 市场集中度：[CR5=XX%]

关键结论：
[该力量对行业盈利能力的影响程度和方向]
```

**检查点**:
- [ ] 是否识别了主要竞争者？
- [ ] 是否有市场份额数据？
- [ ] 是否分析了竞争特征？

### Step 3: 分析新进入者威胁（Threat of New Entrants）

**问题**: 新竞争者进入行业的难度有多大？进入壁垒有多高？

**分析维度**:
- **规模经济**：大规模生产带来的成本优势
- **品牌忠诚度**：现有品牌的客户粘性
- **资本要求**：进入所需的资金投入
- **转换成本**：客户更换供应商的成本
- **分销渠道**：渠道的可获得性
- **政府政策**：准入许可、牌照要求
- **技术壁垒**：专利、技术秘密
- **学习曲线**：经验和效率优势

**格式**:
```
新进入者威胁：

威胁程度：[高威胁/中等威胁/低威胁]

进入壁垒分析：
| 壁垒类型 | 高度 | 具体描述 |
|---------|------|---------|
| 规模经济 | 高/中/低 | ... |
| 品牌忠诚度 | 高/中/低 | ... |
| 资本要求 | 高/中/低 | ... |
| 转换成本 | 高/中/低 | ... |
| 分销渠道 | 高/中/低 | ... |
| 政府政策 | 高/中/低 | ... |
| 技术壁垒 | 高/中/低 | ... |
| 学习曲线 | 高/中/低 | ... |

最近进入者案例：
1. [进入者名称]
   - 进入时间：[日期]
   - 进入方式：[方式描述]
   - 遇到的困难：[困难描述]
   - 当前状态：[成功/失败/挣扎中]

潜在进入者：
1. [潜在进入者]
   - 进入可能性：[高/中/低]
   - 可能的进入方式：[方式描述]
   - 潜在影响：[影响描述]

关键结论：
[该力量对行业盈利能力的影响程度和方向]
```

**检查点**:
- [ ] 是否评估了所有进入壁垒？
- [ ] 是否分析了最近进入者案例？
- [ ] 是否识别了潜在进入者？

### Step 4: 分析替代品威胁（Threat of Substitutes）

**问题**: 有哪些替代产品或服务？替代威胁有多大？

**分析维度**:
- **替代品数量**：替代品的多少
- **替代品价格**：相对于行业产品的价格
- **替代品性能**：功能和质量对比
- **转换成本**：用户转向替代品的成本
- **购买者意愿**：用户接受替代品的意愿

**格式**:
```
替代品威胁：

威胁程度：[高威胁/中等威胁/低威胁]

主要替代品：
1. [替代品1]
   - 替代方式：[完全替代/部分替代]
   - 价格对比：[便宜/相当/昂贵]
   - 性能对比：[更好/相当/更差]
   - 转换成本：[高/中/低]
   - 威胁程度：[高/中/低]

2. [替代品2]
   - 替代方式：...
   - 价格对比：...
   - 性能对比：...
   - 转换成本：...
   - 威胁程度：...

用户替代意愿：
- 价格敏感型用户：[意愿描述]
- 性能敏感型用户：[意愿描述]
- 其他用户：[意愿描述]

关键结论：
[该力量对行业盈利能力的影响程度和方向]
```

**检查点**:
- [ ] 是否识别了所有主要替代品？
- [ ] 是否分析了价格-性能对比？
- [ ] 是否评估了用户替代意愿？

### Step 5: 分析供应商议价能力（Supplier Power）

**问题**: 供应商有多大的议价能力？

**分析维度**:
- **供应商集中度**：供应商数量和规模
- **供应商差异化**：产品/服务的可替代性
- **对供应商的依赖**：采购占比
- **供应商前向整合能力**：供应商能否进入下游
- **转换成本**：更换供应商的成本
- **供应商对行业的依赖**：供应商营收占比

**格式**:
```
供应商议价能力：

议价能力：[高/中/低]

主要供应商：
1. [供应商类型1]
   - 供应商数量：[多/中/少]
   - 集中度：[高/中/低]
   - 产品差异化：[高/中/低]
   - 转换成本：[高/中/低]
   - 对我们依赖：[高/中/低]
   - 议价能力：[高/中/低]

2. [供应商类型2]
   - ...

关键投入品分析：
1. [投入品1]
   - 重要性：[高/中/低]
   - 供应稳定性：[稳定/不稳定]
   - 价格波动：[大/中/小]
   - 替代品：[有/无]

供应商威胁：
- 提价威胁：[高/中/低]
- 降低质量威胁：[高/中/低]
- 断供威胁：[高/中/低]

关键结论：
[该力量对行业盈利能力的影响程度和方向]
```

**检查点**:
- [ ] 是否识别了所有关键供应商类型？
- [ ] 是否分析了供应商集中度？
- [ ] 是否评估了供应商议价能力？

### Step 6: 分析买方议价能力（Buyer Power）

**问题**: 买方（客户）有多大的议价能力？

**分析维度**:
- **买方集中度**：买方数量和规模
- **产品差异化**：产品对买方的重要性
- **买方价格敏感度**：价格占买方成本的比例
- **转换成本**：买方更换供应商的成本
- **买方后向整合能力**：买方能否自己生产
- **买方信息透明度**：买方对市场的了解程度
- **产品对买方产品质量的影响**

**格式**:
```
买方议价能力：

议价能力：[高/中/低]

主要买方类型：
1. [买方类型1]
   - 买方数量：[多/中/少]
   - 集中度：[高/中/低]
   - 采购占比：[占我们营收XX%]
   - 价格敏感度：[高/中/低]
   - 转换成本：[高/中/低]
   - 后向整合能力：[强/中/弱]
   - 议价能力：[高/中/低]

2. [买方类型2]
   - ...

买方威胁：
- 压价威胁：[高/中/低]
- 要求更高质量：[高/中/低]
- 要求更多服务：[高/中/低]
- 更换供应商威胁：[高/中/低]

关键结论：
[该力量对行业盈利能力的影响程度和方向]
```

**检查点**:
- [ ] 是否识别了所有主要买方类型？
- [ ] 是否分析了买方集中度？
- [ ] 是否评估了买方议价能力？

### Step 7: 汇总分析与战略建议

**格式**:
```
五力强度汇总：

| 五力 | 强度 | 对盈利能力影响 | 关键因素 |
|-----|------|--------------|---------|
| 现有竞争者 | 高/中/低 | 负面/中性/正面 | ... |
| 新进入者威胁 | 高/中/低 | 负面/中性/正面 | ... |
| 替代品威胁 | 高/中/低 | 负面/中性/正面 | ... |
| 供应商议价 | 高/中/低 | 负面/中性/正面 | ... |
| 买方议价 | 高/中/低 | 负面/中性/正面 | ... |

行业吸引力评估：
- 综合评分：[1-10分]
- 盈利能力：[高/中/低]
- 吸引力评估：[吸引/一般/不吸引]

竞争战略建议：

   - [策略建议]
```

## 常见错误

- ❌ 行业定义过于宽泛或过于狭窄
- ❌ 只分析现状，不分析趋势变化
- ❌ 五力分析缺乏数据支撑
- ❌ 只做分析，不制定战略建议
- ❌ 忽视力量间的相互影响
- ❌ 静态分析，不考虑动态变化
- ❌ 过度关注现有竞争，忽视潜在威胁
- ❌ 数据来源不可靠

## 与其他方法的关系

- **前置方法**: market-research（市场调研）、industry-analysis（行业分析）
- **后置方法**: swot-analysis（识别优势和劣势）、competitor-analysis（深入竞争分析）
- **并行方法**: pestle-analysis（宏观环境分析）、ansoff-matrix（增长策略）
