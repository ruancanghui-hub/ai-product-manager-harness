# PESTLE Analysis (PESTLE分析)

## 目的

系统化扫描宏观环境中的六大因素（Political政治、Economic经济、Social社会、Technological技术、Legal法律、Environmental环境），识别外部变化对组织的影响，为战略决策提供环境背景。

## 核心概念

```
┌─────────────────────────────────────────────────────────────────┐
│                        PESTLE 分析                               │
├─────────────┬─────────────┬─────────────┬─────────────────────┤
│   P 政治因素  │   E 经济因素  │   S 社会因素  │                     │
│  Political   │  Economic   │   Social    │                     │
│             │             │             │   外部环境            │
│  政府政策    │  经济增长    │  人口结构    │   不可控因素          │
│  政治稳定性  │  利率/汇率   │  文化趋势    │   长期趋势            │
│  贸易政策    │  通胀/失业率  │  生活方式    │                     │
├─────────────┼─────────────┼─────────────┤                     │
│   T 技术因素  │   L 法律因素  │   E 环境因素  │                     │
│Technological │   Legal     │Environmental│                     │
│             │             │             │                     │
│  技术创新    │  劳动法      │  气候变化    │                     │
│  研发投入    │  消费者保护  │  环保法规    │                     │
│  技术变革速度 │  知识产权    │  可持续发展  │                     │
└─────────────┴─────────────┴─────────────┴─────────────────────┘
```

**分析层次**:
- **宏观趋势**：全球/国家层面的长期趋势
- **行业影响**：对特定行业的影响
- **组织影响**：对本组织的具体影响

## 何时使用

- ✅ 战略规划前，了解宏观环境
- ✅ 新市场进入前，评估环境风险
- ✅ 投资决策前，分析外部因素
- ✅ 长期趋势预测，识别机会和威胁
- ✅ 配合SWOT分析，作为外部环境输入
- ✅ 跨国业务决策，评估各国环境差异

## 使用步骤

### Step 1: 明确分析范围

**问题**: 分析的地理范围、时间范围和行业范围是什么？

**格式**:
```
分析对象：[公司/产品/项目]
地理范围：[全球/国家/区域/城市]
时间范围：[短期1年/中期3-5年/长期5-10年]
行业范围：[行业描述]
分析目的：[战略规划/市场进入/投资决策...]
```

**检查点**:
- [ ] 地理范围是否明确？
- [ ] 时间范围是否合理？
- [ ] 分析目的是否清晰？

### Step 2: 分析政治因素（Political）

**问题**: 政治环境和政府政策如何影响我们？

**分析维度**:
- **政府政策**：产业政策、税收政策、补贴政策
- **政治稳定性**：政权更替、社会稳定
- **贸易政策**：关税、贸易壁垒、贸易协定
- **外交关系**：国际关系、制裁
- **政府效率**：审批流程、腐败程度

**格式**:
```
政治因素 (Political)：

有利因素：
1. [因素1]
   - 具体描述：[政策/事件]
   - 影响程度：[高/中/低]
   - 时间紧迫性：[立即/短期/长期]

不利因素：
1. [因素1]
   - 具体描述：[政策/事件]
   - 影响程度：[高/中/低]
   - 应对策略：[简述]

关键变化趋势：
- [趋势1]：[过去→现在→未来预测]
```

**检查点**:
- [ ] 是否涵盖国家和地方政策？
- [ ] 是否考虑政策变化趋势？
- [ ] 是否识别了有利和不利因素？

### Step 3: 分析经济因素（Economic）

**问题**: 经济环境如何影响我们的业务？

**分析维度**:
- **宏观经济**：GDP增长、经济周期
- **货币政策**：利率、汇率、通胀率
- **劳动力市场**：工资水平、失业率、人才供给
- **消费者经济**：可支配收入、消费信心、储蓄率
- **资本市场**：融资环境、投资者信心

**格式**:
```
经济因素 (Economic)：

有利因素：
1. [因素1]
   - 具体数据：[经济指标]
   - 对业务影响：[正面影响]
   - 持续时间：[短期/中期/长期]

不利因素：
1. [因素1]
   - 具体数据：[经济指标]
   - 对业务影响：[负面影响]
   - 应对策略：[简述]

关键指标趋势：
- GDP增长率：[过去3年数据] → [未来预测]
- 通胀率：[过去3年数据] → [未来预测]
- 利率：[过去3年数据] → [未来预测]
- 汇率：[过去3年数据] → [未来预测]
```

**检查点**:
- [ ] 是否使用了权威数据源？
- [ ] 是否区分了短期波动和长期趋势？
- [ ] 是否分析了正负面影响？

### Step 4: 分析社会因素（Social）

**问题**: 社会文化和人口变化如何影响我们？

**分析维度**:
- **人口结构**：年龄结构、人口增长、城市化率
- **文化趋势**：价值观变化、生活方式变化
- **教育水平**：受教育程度、技能水平
- **消费习惯**：消费偏好、消费渠道变化
- **社会问题**：收入差距、社会流动性

**格式**:
```
社会因素 (Social)：

有利因素：
1. [因素1]
   - 具体描述：[社会趋势]
   - 影响人群：[目标用户]
   - 影响程度：[高/中/低]

不利因素：
1. [因素1]
   - 具体描述：[社会趋势]
   - 影响人群：[目标用户]
   - 应对策略：[简述]

关键趋势：
- 人口结构：[老龄化/年轻化/城市化...]
- 生活方式：[远程办公/健康意识/环保意识...]
- 消费习惯：[线上化/个性化/体验式消费...]
```

**检查点**:
- [ ] 是否基于人口统计数据？
- [ ] 是否考虑了文化差异？
- [ ] 是否识别了新兴趋势？

### Step 5: 分析技术因素（Technological）

**问题**: 技术变革如何影响我们的行业和业务？

**分析维度**:
- **新兴技术**：AI、区块链、IoT、5G等
- **技术成熟度**：技术发展阶段、应用成熟度
- **研发投入**：行业研发投入、政府研发支持
- **技术替代**：颠覆性技术、技术淘汰
- **基础设施**：网络基础设施、技术生态

**格式**:
```
技术因素 (Technological)：

机会因素：
1. [技术1]
   - 技术成熟度：[成熟/成长/萌芽]
   - 应用场景：[具体场景]
   - 对业务影响：[正面影响]
   - 时间窗口：[何时可用]

威胁因素：
1. [技术1]
   - 技术描述：[颠覆性技术]
   - 威胁程度：[高/中/低]
   - 时间紧迫性：[立即/短期/长期]
   - 应对策略：[简述]

关键技术趋势：
- [技术领域1]：[现状] → [未来预测]
- [技术领域2]：[现状] → [未来预测]
```

**检查点**:
- [ ] 是否识别了颠覆性技术？
- [ ] 是否评估了技术成熟度？
- [ ] 是否考虑了技术应用时间线？

### Step 6: 分析法律因素（Legal）

**问题**: 法律法规如何影响我们的业务？

**分析维度**:
- **行业法规**：行业准入、经营许可
- **劳动法规**：用工法规、劳动合同
- **消费者保护**：隐私保护、消费者权益
- **知识产权**：专利、版权、商标
- **合规要求**：数据合规、反垄断、反腐败

**格式**:
```
法律因素 (Legal)：

有利因素：
1. [法规1]
   - 具体内容：[法规描述]
   - 生效时间：[日期]
   - 对业务影响：[正面影响]

不利因素/合规要求：
1. [法规1]
   - 具体内容：[法规描述]
   - 合规成本：[预估成本]
   - 合规期限：[截止日期]
   - 不合规风险：[风险描述]

关键法规变化：
- [法规领域1]：[现状] → [新规] → [影响]
- [法规领域2]：[现状] → [新规] → [影响]
```

**检查点**:
- [ ] 是否涵盖了所有相关法规？
- [ ] 是否识别了新法规变化？
- [ ] 是否评估了合规成本？

### Step 7: 分析环境因素（Environmental）

**问题**: 环境问题和可持续发展如何影响我们？

**分析维度**:
- **气候变化**：碳排放、极端天气
- **环保法规**：环保标准、排放限制
- **可持续发展**：ESG要求、绿色消费
- **资源稀缺**：能源、水资源、原材料
- **环保意识**：消费者环保意识、投资者ESG关注

**格式**:
```
环境因素 (Environmental)：

机会因素：
1. [因素1]
   - 具体描述：[环保趋势/法规]
   - 对业务影响：[正面影响]
   - 时间窗口：[短期/中期/长期]

威胁因素：
1. [因素1]
   - 具体描述：[环境风险]
   - 影响程度：[高/中/低]
   - 应对策略：[简述]

关键环境趋势：
- 气候变化：[碳排放目标/极端天气风险]
- 环保法规：[新环保标准/排放限制]
- ESG要求：[投资者/消费者要求]
```

**检查点**:
- [ ] 是否考虑了气候变化影响？
- [ ] 是否识别了环保法规要求？
- [ ] 是否评估了ESG影响？

### Step 8: 汇总分析与策略建议

**格式**:
```
PESTLE汇总矩阵：

| 维度 | 关键因素 | 影响程度 | 机会/威胁 | 时间紧迫性 | 应对策略 |
|------|---------|---------|----------|-----------|---------|
| 政治  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |
| 经济  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |
| 社会  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |
| 技术  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |
| 法律  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |
| 环境  | ...     | 高/中/低 | O/T      | 立即/短期/长期 | ...    |

优先关注事项（Top 5）：
1. [事项1] - [维度] - [应对策略]
2. ...

战略建议：
1. [建议1]：[具体行动]
2. ...
```

## 常见错误

- ❌ 列出太多因素，没有优先级
- ❌ 因素过于泛泛，缺乏具体数据
- ❌ 只做分析，不制定应对策略
- ❌ 忽视因素间的相互影响
- ❌ 分析静态化，不考虑趋势变化
- ❌ 地理范围不明确
- ❌ 混淆内部和外部因素
- ❌ 数据来源不可靠
